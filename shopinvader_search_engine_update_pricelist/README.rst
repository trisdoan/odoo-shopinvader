This module extends the functionality of the
``shopinvader_search_engine_update`` to mark the information exported to
the search engine as to be recomputed when the product pricelist data
are updated.

Use Cases / Context
===================

When you update your product pricelist data and this information is
exposed by the search engine, you expect to see your changes applied
into the search engine as soon as possible.
